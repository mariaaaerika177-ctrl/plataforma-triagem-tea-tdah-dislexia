# 🧠 Plataforma de Triagem Educacional

Projeto acadêmico desenvolvido com o objetivo de propor uma solução tecnológica para triagem inicial de sinais de TEA, TDAH e dislexia, com foco em inclusão educacional e acessibilidade.

---

## 🚀 Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  

---

## 🎯 Objetivo
Desenvolver uma plataforma acessível para auxiliar na identificação inicial de necessidades educacionais específicas e promover inclusão social.

---

## ⚙️ Funcionalidades propostas
- Interface web responsiva  
- Simulação de triagem inicial  
- Conteúdos educativos  
- Foco em acessibilidade (WCAG)  

---

## 🧠 Aprendizados
- Engenharia de requisitos  
- Modelagem de sistemas (UML)  
- Desenvolvimento front-end  
- Princípios de usabilidade e acessibilidade  

---

## 📌 Status do projeto
Projeto acadêmico (não implementado em produção)
